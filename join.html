Instructions to hack this code on a Windows PC:

The source code is held on GitHub, which has a feature whereby simple html and javascript pages can be automatically published on a GitHub url. Free versions of both these features are used to host this simple JavaScript powered site. The site and its games can be run from the following URL https://nvanraalte.github.io/javascriptgames.
  1. You will need to install GitBash. This is a command line program that runs on Windows and will let you get and update the source code.
    • Download GitBash from https://gitforwindows.org.
    • Run the download file to install GitBash. The default settings will be fine, but if you spot the one to enable symlinks, it will be useful to check this one.
  2. Next you will need to decide where you want to put the source code files on your local Windows PC to hack them.
    • In your Windows explorer, choose a directory where you want the files to go - you may want to create a new folder to put them in.
    • Right click your chosen folder and, having installed GitBash, you should see an option 'Git Bash here' - select this to launch the GitBash command line tool.
    • Type (or cut and paste) the following and click enter:
      git checkout https://github.com/nvanraalte/javascriptgames
    • Once that is done use the following GitBash command to go into the new 'javascriptgames' directory:
      cd javascriptgames
    • GitBash now probably shows the directory name with a ' (master)' at the end. Type the following command to change to the area (the name in the brackets) to the one you can use to upload your improvements to:
      git checkout develop
    • In GitBash the directory should now have a ' (develop)' at the end.
  3. To update the code you may need a GitHub account. The GitHub website is https://github.com.
